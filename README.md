ECG Signal Processor

本项目是一个基于医学信息工程背景开发的数字信号处理流水线，专注于心电信号（ECG）的自动化降噪与特征提取。

技术栈：Python / NumPy / SciPy / Matplotlib / WFDB

功能特性

在线或本地读取并解析 MIT-BIH Arrhythmia Database 的生理信号数据。
运用零相移带通滤波，消除信号基线漂移与高频肌电噪声。
应用 50Hz 陷波器（Notch Filter），精准滤除工频干扰。
部署波峰检测算法以定位 R 波，并基于 R-R 间期计算心率。

技术亮点

在临床采集环境中，硬件设备不可避免地会引入 50Hz 的工频干扰，陷波器的应用能够在不引起心电波形畸变的前提下，精准且彻底地抑制该特定频率的噪声。此外，高鲁棒性的 R 波检测是整个心电分析系统的核心，它不仅为连续心率的计算提供了准确的时间锚点，也为后续的心率变异性（HRV）分析和病理诊断奠定了数据基础。

快速开始

安装所需依赖包

运行信号处理主程序
python main.py
